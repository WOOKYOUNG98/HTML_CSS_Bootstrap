인라인 요소 - 글씨를 다루는 용도
인라인 요소는 최대한 글자에 사이즈를 딱 맞추려고함
인라인 요소는 css속성을 사용해도 크기를 조정할수없다.
ex) <span>

<span><div></div></span>  = 불가능
- 인라인 요소안에 블록 요소를 넣을 수 없음
- 즉, 글자 안에 상자를 넣을 수 없다.
<span><span></span></span> = 가능

블록 요소 - 상자를 만들기 위함
부모 요소의 크기만큼 최대한 늘어나려고 함(가로길이)
포함된 콘텐츠 크기만큼 자동으로 줄어드려고함(세로길이)
ex) <div>
대표적인 블록요소!
본질적으로 아무것도 나타내지 않는, 콘텐츠 영역을 설정하는 용도
<div><div></div></div> = 가능
<div><span></span></div> = 가능
- 블록요소 안에 인라인 요소를 넣을 수 있다.
- 즉 상자안에 상자 또는 상자안에 글자를 넣는게 가능하다.

<html></html>
- 문서의 전체 범위
- HTML 문서가 어디에서 시작하고, 어디에서 끝나는지 알려주는 역할

 <head></head>
- 문서의 정보를 나타내는 범위
- 웹 브라우저가 해석해야 할 웹 페이지의 제목, 설명, 사용할 파일 위치,
스타일(css)같은, 웹 페이지의 보이지 않는 정보를 작성하는 범위이다.

<body></body>
- 문서의 구조를 나타내는 범위
- 사용자 화면을 통해 보여지는 로고, 헤더, 푸터, 내비게이션, 메뉴,
버튼, 이미지 같은 웹 페이지의 보여지는 구조를 작성하는 범위이다.

<!DOCTYPE html>
- 문서의 HTML 버전을 지정
